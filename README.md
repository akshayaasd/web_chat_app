# Multi-Room Chat Application

A real-time, multi-room chat application built with **FastAPI** (Python) and **React** (Vite).

## Features
- Real-time messaging using WebSockets.
- Multiple chat rooms with automated room management.
- Persistent message storage with MongoDB.
- Modern, responsive UI.

## Tech Stack
- **Frontend**: React, Vite, CSS.
- **Backend**: FastAPI, Motor (Async MongoDB), WebSockets.
- **Database**: MongoDB.

## Getting Started

### Prerequisites
- Python 3.8+
- Node.js & npm
- MongoDB (Local or Atlas)

### Backend Setup
1. Navigate to the `backend` directory.
2. Install dependencies (recommended: create a virtual environment).
   ```bash
   pip install fastapi uvicorn motor
   ```
3. Run the server:
   ```bash
   python main.py
   ```

### Frontend Setup
1. Navigate to the `frontend` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
